## 📖 Sobre o Projeto

Este projeto foi desenvolvido como **Projeto Final do Curso de Analista de Dados do SENAC**.

O objetivo foi realizar uma análise completa de um conjunto de dados de vendas de uma empresa fictícia que dei o nome de (**SuperLoja 360**) devido a sua variedade de produtos, aplicando todas as etapas de um projeto de Data Analytics, desde o tratamento dos dados até a geração de insights estratégicos para apoio à tomada de decisão.

O trabalho contempla:

- Limpeza e preparação dos dados;
- Análise exploratória;
- Modelagem dos dados;
- Desenvolvimento de Dashboard Executivo no Power BI;
- Elaboração de Relatório Analítico;
- Geração de recomendações para o negócio.

---

# 🎯 Objetivo

Transformar dados transacionais em informações estratégicas capazes de responder perguntas importantes do negócio, como:

- Qual categoria gera maior faturamento?
- Existe sazonalidade nas vendas?
- Quais produtos sustentam a receita?
- Como os clientes se comportam?
- Quais estados possuem maior potencial?
- Como otimizar campanhas e estoque?

# 📈 Principais Insights

## 📌 Receita concentrada

A categoria **Eletrônicos** representa aproximadamente **79% do faturamento**, indicando alta dependência desse segmento. 

---

## 📌 Sazonalidade

Foram identificados picos relevantes de faturamento nos meses de:

- Fevereiro
- Março
- Setembro

Já abril apresentou o menor desempenho do período. 

---

## 📌 Produtos estratégicos

Quatro produtos concentram aproximadamente **76% da receita anual**:

- Notebook
- Smartphone
- Tablet
- Smartwatch

Esses itens representam o principal núcleo financeiro da operação.

---

## 📌 Formas de pagamento

Os pagamentos via:

- Cartão de Crédito
- Pix

representam mais de **76% das transações**, indicando rapidez no recebimento, mas também dependência operacional. 

---

## 📌 Distribuição Geográfica

Os estados com maior faturamento foram:

- Goiás
- São Paulo
- Paraná
- Ceará
- Bahia

Apesar disso, nenhuma região domina o mercado, indicando oportunidades para campanhas regionais.

---

## 📌 Perfil dos Clientes

A base analisada apresentou:

- 493 clientes ativos
- Ticket médio de R$ 1.365
- Avaliação média de 4,08/5

Esses indicadores sugerem uma operação madura, com espaço para melhoria na fidelização. 

---

# 💡 Recomendações

Com base nos resultados, foram propostas cinco ações estratégicas:

- Garantir disponibilidade dos produtos de maior faturamento;
- Diversificar o portfólio para reduzir concentração;
- Planejar campanhas considerando a sazonalidade;
- Incentivar pagamentos via Pix;
- Desenvolver campanhas regionais direcionadas.

Essas recomendações têm potencial para proteger a receita atual e impulsionar o crescimento da empresa.
